# منظومة نور — تحضير دروس اللغة العربية
## (Noor Arabic Prep System)

منصة احترافية متكاملة لمعلمي اللغة العربية في اليمن، مصممة للعمل محلياً بكفاءة عالية مع مراعاة أحدث المعايير التربوية.

### 🚀 المميزات الرئيسية
- **العمل دون إنترنت (Offline-first)**: يعتمد النظام على `IndexedDB` لتخزين كافة البيانات في المتصفح محلياً.
- **توافق المنهج اليمني**: يدعم الصفوف (7، 8، 9) مع كافة الأقسام (النحو، النصوص، القراءة، الإملاء، الخط، التعبير).
- **استراتيجيات جيل ألفا**: محرك ذكي يدمج استراتيجيات تعليمية مخصصة لجيل ألفا وخصائصه المعرفية.
- **تصدير احترافي**: دعم كامل لتصدير الخطط بصيغة Microsoft Word وجاهزية تامة للطباعة الورقية.
- **الأمان والنسخ الاحتياطي**: ميزة تصدير واستيراد البيانات (JSON) لضمان عدم فقدان العمل عند تغيير الجهاز أو تنظيف المتصفح.
- **التغذية الراجعة التطورية**: حل تحليل إحصائي لمستوى تنوع التحضير والالتزام بالمنهج.

### 🏗️ المعمارية التقنية (Architecture)
- **Frontend**: React 19 + TypeScript + Vite.
- **Styling**: Tailwind CSS 4.
- **Animations**: Framer Motion.
- **Database**: IndexedDB (Native Web API).
- **Data Layers**: فصل كامل بين منطق بناء الخطة (Engine) وقسم البيانات (Constants).

### 📂 هيكلية المشروع
```text
/src
  /components     — المكونات الواجهة (Form, Preview, Backup, Sidebar)
  /constants      — الثوابت التربوية والمحتوى المنهجي
  /hooks          — إدارة الحالة والارتباط بقاعدة البيانات
  /lib            — المحركات التقنية (DB, Engine, Utils)
  /services       — منطق التعامل مع البيانات الخارجية مستقبلاً
  types.ts        — تعريفات الأنواع والواجهات (TypeScript Interfaces)
  App.tsx         — المكون المركزي وهيكل التنقل
```

### 🛠️ تشغيل النظام محلياً
1. تثبيت الاعتمادات: `npm install`
2. التشغيل: `npm run dev`
3. البناء للإنتاج: `npm run build`

### 🔒 الخصوصية والأمان
يعمل النظام محلياً بنسبة 100%. لا يتم إرسال أي بيانات (أسماء المعلمين، المدارس، الخطط) إلى أي خادم خارجي. أنت تملك بياناتك وتتحكم بمكان تخزينها.
